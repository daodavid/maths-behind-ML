def sigmoid(w,b,range=10):
    """
    
    """
        
    f = lambda x : 1/(1+np.exp(-x))
    x = np.linspace(-range,range)
    x = np.array([x,x]).T
    z = np.array( [ np.array(w).dot(i)+b for i in x])
    result = np.array([f(i) for i in z])
    return z,result

The above graphics is called cross-etropy function , and also called Log Likehood function which can be used as cost function for logistic regression¶

The logistic regression cost function for all input variable can be rewritten as :

$$ (1) J(\Theta) = \sum_i^mCost(h_{\Theta}(x_i),y_i)=-\frac{1}{m}\big(\sum_i^my_i.log(h_{\Theta}(x_i)) + (y_i-1)log(1 - h_{\Theta}(x_i))\big)$$

How we can find the min entropy ? What does mean min entropy ?

From graphic 1 let notice that if $$ (3) \; \; \; x \to 1 \; then \; -log(\sigma(x)) \to 1 $$ $$ (4) \; \; \; x \to 1 \; then \; -log(\sigma(x)) \to 1 \; $$

$$(5) \; \; \;x \to 0 \; then \; log(\sigma(x)) \to \infty $$ $$ (6) \; \; \;x \to 0 \; then \; -log(\sigma(x)) \to -\infty $$

from eq. (3) we can conclude that when we have to fit label(taget) value which is equal to 1, then eq. $-log(\sigma(x))$ must be minimize and this give us the min error also called min-entropy, if we use eq.(4) we have to be maximize $log(\sigma(x))$ this is also be give the min-entropy but $log(\sigma(x))$ has not the close form for findind extremum that fact makes it dificult ,but the both functions is the opposite. If we want to fit taget value = 0 both functions is useless because the functions log(1(x)),log(1(x))+ ,they also have close form and the error+ and entropy as well

From graphic 2 let notice that if $$ (7) \; \; \; x \to 1 \; then \; -log(1-\sigma(x)) \to \infty $$ $$ (8) \; \; \; x \to 1 \; then \; -log(1-\sigma(x)) \to \infty; $$

$$(9) \; \; \;x \to 0 \; then \; \;log(1-\sigma(x)) \to 0 $$ $$ (10) \; \; \;x \to 0 \; then \; \; -log(1-\sigma(x)) \to 0 $$

The above thoughts can be taken into account for eq. (7)(8)(9)(10) as well, but with the opposite logic, reaching that when we have to fit the target value equal=1 then the minimization of $-log(1-\sigma(x))$ will give satisfy result

Briefly, as far as well we saw that the function $-log(\sigma(x))$ is the best choice for cost function when the target value is equal to 1 and when the target value is equal to 0 we could use the $-log(1-\sigma(x))$ and the equation $$ Cost(h_{\Theta}(x),y) = -y.log(h_{\Theta}(x)) - (y-1)log(1 - h_{\Theta}(x))$$ contains both functions together , and when Y=0 it remain only $log(h_{\Theta}(x))$ and when y=1 the function remains $log(1 - h_{\Theta}(x))$
Our purpose is to minimize $ Cost(h_{\Theta}(x),y)$ where

$$ h_{\theta(x)}=\sigma(\vec{\theta}.\vec{x})=\frac{1}{1+e^{\vec{\theta}.\vec{x}}}=\frac{1}{1+e^{z}}$$

which involve the task for finding the best parameters of $\vec{\Theta} = [\theta_n ,..., \theta_1 ,\theta_0]$

Optimization problem¶

We will use the well known Gradient denscent for optimization.Our goal is to go find : $$\frac{\partial J(\Theta)}{\partial x_i}$$

$$ J(\Theta) = \sum_i^mCost(h_{\Theta}(x_i),y_i)=-\frac{1}{m}\big(\sum_i^my_i.log(h_{\Theta}(x_i)) + (1-y_i)log(1 - h_{\Theta}(x_i))\big)$$

let to find some derivates of function wich will be very useful


$$(1)\; \; \frac{\partial h_{\Theta(z)}}{\partial x_i } = \frac{d h_{\Theta(z)}}{dz}.\frac{\partial z}{\partial x_i} $$



$$ (2)\; \; \frac{d h_{\Theta(z)}}{dz} = \frac{\frac{1}{1+e^{-z}}}{dz}= \frac{e^{-z}}{(1+e^{-z})^2}=\frac{1}{1+e^{-z}}(1 -\frac{1}{1+e^{-z}} ) = h_{\Theta}(z)(1 - h_{\Theta}(z)) $$



$$ (3) \; \ \frac{dlog(h_{\Theta}(x_i))}{dz} = \frac{1}{h_{\Theta}(z)}\frac{dh_{\Theta}(z))}{dz} =\frac{1}{h_{\Theta}(z)} h_{\Theta}(z)(1 - h_{\Theta}(z))=(1 - h_{\Theta}(z)) $$



$$ (4) \; \;\frac{dlog(1 - h_{\Theta}(z))}{dz} = \frac{1}{(1- h_{\Theta}(z))}\frac{d(1-h_{\Theta}(z)))}{dz} = \frac{1}{(1- h_{\Theta}(z))}\frac{-d(h_{\Theta}(z)))}{dz}=-\frac{1}{(1-h_{\Theta}(z))} h_{\Theta}(z)(1 - h_{\Theta}(z))=- h_{\Theta}(z)) $$

Let to find partial derivate of function $J(\Theta)$ respect to $x_i$

$$(5) \; \; \dot{J}(\theta)_{\theta_i}=\frac{\partial J(\theta)}{\partial \theta_i} = -\frac{1}{m}\sum_i^m\big(y_i \frac{\partial log(h_{\Theta}(z))}{\partial \theta_i}+ (1-y_i) \frac{\partial log(h_{\Theta}(z)-1)}{\partial \theta_i}\big) = $$
$$-\frac{1}{m}\sum_i^m\big(y_i \frac{dlog(h_{\Theta}(z))}{dz}\frac{\partial z}{\partial \theta_i}+ (y_i-1) \frac{dlog(1- h_{\Theta}(z))}{dz}\frac{\partial z}{\partial \theta_i}\big) $$

appling eqations (3) (4) to (5) we achieve $$\dot{J}(\theta)_{.\theta_i}=-\frac{1}{m}\sum_i^m\big(y_i (1 - h_{\Theta}(z))\frac{\partial z}{\partial .\theta_i}+ (1-y_i) (- h_{\Theta}(z))\frac{\partial z}{\partial x_i})\big) = -\frac{1}{m}\sum_i^m\big(y_i\frac{\partial z}{\partial x_i} -y_ih_{\Theta}(z))\frac{\partial z}{\partial \theta_i} +y_ih_{\Theta}(z))\frac{\partial z}{\partial \theta_i} - h_{\Theta}(z)\frac{\partial z}{\partial \theta_i}) $$

$$6) \; \; \dot{J}(\theta)_{\theta_i} = -\frac{1}{m}\sum_i^m \big(y_i \frac{\partial z}{\partial \theta_i} - h_{\Theta}(z)\frac{\partial z}{\partial \theta_i}\big) = -\frac{1}{m}\sum_i^m \big(y_i - h_{\Theta}(z)\big)\frac{\partial z}{\partial \theta_i} $$


where $$ (7) \;\frac{\partial z}{\partial \theta_i}=\frac{\partial \theta_n x_n + ...\theta_1 x_1+\theta_0}{\partial \theta_i } = x_i $$
$$and \; for\; x_0=1\; \frac{\partial z}{\partial \theta_0} =1 $$
applying eq(7) into (6) we achieve the final result :

$$(12)\; \; \frac{\partial J(\theta)}{\partial \theta_i} =-\frac{1}{m}\sum_i^m \big(y_i - h_{\Theta}(z)\big )x_i $$

Gradient descent alogorith must to use implemnt eq.(12) wich will look like this

$$ 13) \; \; \theta_i = \theta_i - \frac{\partial J(\theta)}{\partial a} <=>$$



$$\; \; \theta_i = \theta_i - (-\frac{1}{m}\sum_i^m \big(y_i - h_{\Theta}(z)\big )x_i) <=> $$
$$\theta_i = \theta_i - \frac{1}{m}\sum_i^m \big( h_{\Theta}(z) - y_i\big )x_i $$

Our gradient descent algorithm is :¶

$$\theta_i = \theta_i - \frac{1}{m}\sum_i^m \big( h_{\Theta}(z) - y_i\big )x_i $$

for intersept $\theta_0$ $$\theta_0 = \theta_0 - \frac{1}{m}\sum_i^m \big( h_{\Theta}(z) - y_i\big )$$
